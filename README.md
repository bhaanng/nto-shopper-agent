# 🌍 World Market Shopping Agent

An AI-powered shopping assistant for World Market, built with Claude 4 Opus and Python.

**Mira** helps customers discover globally-inspired furniture, home décor, kitchen items, international foods, and unique finds from World Market's catalog.

## 🚀 Features

- **Intelligent Product Search** - Natural language queries with style, room, and budget understanding
- **Global Style Expertise** - Recommends products based on bohemian, modern, traditional, coastal, and other aesthetics
- **Smart Filtering** - Category, price range, and keyword search
- **Conversational UI** - Chat-based interface with follow-up questions and suggestions
- **Web Interface** - Clean Streamlit UI for easy interaction

## 🛠️ Setup

### Option 1: Run Locally (Quick Start)

1. **Install dependencies**:
   ```bash
   cd world-market-agent
   pip install -r requirements.txt
   ```

2. **Scrape World Market catalog** (first time only):
   ```bash
   cd scraper
   python world_market_scraper.py
   ```
   This will create `data/products.json` with ~140 products (20 per category).

3. **Run the web UI**:
   ```bash
   cd ui
   streamlit run app.py
   ```

4. **Open in browser**: http://localhost:8501

### Option 2: Run with Docker (Best for Sharing)

1. **Build and run**:
   ```bash
   cd world-market-agent
   docker-compose up --build
   ```

2. **Open in browser**: http://localhost:8501

3. **Share with colleagues**:
   - Share the `world-market-agent` folder
   - They just run: `docker-compose up`
   - Done! 🎉

## 🔧 Configuration

### API Key
Set your Anthropic API key in `.env`:
```
ANTHROPIC_API_KEY=sk-your-key-here
```

### Scraper Settings
Edit `scraper/world_market_scraper.py` to adjust:
- `max_products_per_category`: Number of products to scrape per category (default: 20)
- `categories`: Add/remove categories to scrape

### Agent Model
Edit `agent/world_market_agent.py`:
```python
self.model = "claude-opus-4-6"  # Change to claude-sonnet-4-6 for faster/cheaper
```

## 📊 How It Works

1. **User Query** → Streamlit UI captures natural language question
2. **Agent Planning** → Claude creates a TODO plan with search strategy
3. **Tool Execution** → Agent searches product catalog with multiple query variants
4. **Response Generation** → Claude formats results as Markdown with product tables
5. **Follow-up** → Suggests next questions and actions

## 🔍 System Prompt

The agent uses a detailed system prompt adapted from the WeatherTech example:
- **Format 1**: Tool calling with JSON structure
- **Format 2**: Final response with Markdown tables, follow-ups, and suggestions
- **Tools**: `create_todo`, `search_world_market_products`, `get_world_market_product_details`

See `agent/system_prompt.py` for the full prompt.

## 🌐 Scraping More Products

To get a fuller catalog:


⚠️ **Note**: Be respectful of World Market's servers. Add delays between requests and don't scrape excessively.

## 🐛 Troubleshooting

### "Catalog not found" error
Run the scraper first:
```bash
cd scraper
python world_market_scraper.py
```

### Scraper returns 0 products
World Market's HTML may have changed. Check `world_market_scraper.py` and update CSS selectors in `_extract_product_info()`.

### API errors
Verify your API key in `.env`:
```bash
echo $ANTHROPIC_API_KEY  # Should print your key
```
